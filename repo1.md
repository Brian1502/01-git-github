<h3>2. Konfigurasi Github</h3>
<p>Langkah untuk konfigurasi Git sebagai berikut:</p>
<p>1. Buka CMD di komputer/laptop</p>
<p>2. Lalu ketik "git config --global user.name "Nama Anda di GitHub"", ini berguna untuk memasukkan username pada git, setelah itu enter.</p>
<p>3. Lalu ketik "git config --global user.email email@domain.tld", ini digunakan untuk memasukkan email pada git, setelah itu enter.</p>
<p>4. Untuk melihat konfigurasi yang sudah ada ketik"git config --list", lalu enter.</p>
<p>5. Ini adalah contoh hasil konfigurasi pada perubahan email dan nama pada Git.</p>
<img height="350" width="600" src="https://user-images.githubusercontent.com/91442745/135710957-a05a05dd-3722-4360-baf9-b92a2e8855a0.PNG">
</br>
<h3>3. Mengelola Repo</h3>
<p>Langkah-langkah untuk membuat repo sendiri yaitu :</p>
<p>1. Pertama buka github.com lalu jika tidak mempunyai akun harus membuat terlebih dahulu, setelah itu login.</p>
<p>2. Lalu klik tanda + pada bagian atas, setelah itu pilih New repository</p>
<img height="350" width="600" src="https://user-images.githubusercontent.com/91442745/135711208-76b441a3-402c-4c4c-8977-e7b907466a92.PNG">
<p>3. Setelah itu isi nama repo, deskripsi dan centang jika menu bawahnya diperlukan</p>
<img height="500" width="650" src="https://user-images.githubusercontent.com/91442745/135711275-df476c1f-11f4-4146-8fed-95cb0b1a458f.PNG">
<p>4. Lalu klik Create Repository</p>
</br>
<h3>4. Clone Repo</h3>
<p>Proses clone adalah proses untuk menduplikasikan remote repo di GitHub ke komputer lokal. Untuk melakukan proses clone digunakan langkah-langkah yaitu :</p>
<p>1. Pertama buka Github dan login terlebih dahulu.</p>
<p>2. Lalu pilih lokasi penyimpanan/folder untuk penyimpanan clone repo yang akan disimpan.</p>
<p>3. Klik kanan lalu klik Git Bash Here pada folder yang akan dipilih </p>
<p>4. Lalu ketik git clone (link repo)</p>
